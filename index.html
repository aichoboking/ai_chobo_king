클로드 코드를 제대로 쓰려면 좋은 IDE가 필요합니다.
코딩이 처음이어도 괜찮아요. 차근차근 따라해보세요.

시작 전에 — 이게 뭔가요?
🖥️
Antigravity
구글에서 만든 에이전트형 IDE예요. 내 파일들이 어떻게 연결되어 있는지 한눈에 보이고, 오류도 바로 확인할 수 있어요.
→ AI 동료가 일할 수 있는 작업실
⚡
Claude Code
Anthropic이 만든 에이전트형 코딩 도구예요. 내가 원하는 걸 말하면 스스로 코드를 짜고, 파일을 수정하고, 실행까지 해줘요.
→ 작업실 안에서 일하는 AI 동료
🛠️
IDE (통합 개발 환경) 가 없으면?
터미널(검은 화면)로만 작업해야 해요. 파일 구조가 한눈에 안 보이고, Claude Code가 뭘 하는지 따라가기도 어렵고, 오류를 바로 확인하기도 힘들어요.
→ 조금이라도 복잡한 작업이라면 IDE는 필수예요!
→ 기능이 있는 서비스, AI 직원 고용, MD / Skill File 만들기엔 IDE가 필수!
기능 Antigravity 단독 Claude Code 단독 둘 다 사용 ✦

Antigravity 다운로드
아래 링크로 접속해서 내 컴퓨터 환경에 맞는 파일을 받아주세요.

2
설치 진행 (윈도우 기준)
설치 창이 뜨면 아무것도 바꾸지 말고 다음 → 다음 → 다음만 눌러주세요.
⚠️ 설치 경로에 한글이 없는 경로를 권장합니다

설치 전에 먼저 이렇게 해주세요:
내 파일 → 내 PC → 로컬디스크 C에 dev라는 폴더를 만들고, 설치할 때 그 폴더를 선택하세요.

✅ 추천 경로: C:\dev
❌ 피해야 할 경로: C:\Users\홍길동\바탕화면

경로에 한글이 포함되면 나중에 오류가 생기는 경우가 종종 있어요.
설치가 끝나면 본인 구글 계정으로 로그인해주세요.
로컬 디스크 C에 dev 폴더가 생성된 모습
3
프로젝트 폴더 생성
Antigravity가 설치된 폴더와 같은 위치에 claude_project 폴더를 생성합니다. (폴더명은 편한대로)
📁 dev
  ├── 📁 Antigravity    ← 설치된 폴더
  └── 📁 claude_project  ← 새로 만든 폴더
로컬 디스크 C에 dev 폴더가 생성된 모습
4
Claude Code 설치
먼저 Antigravity를 열고, Open Folder에서 아까 만든 claude_project 폴더를 선택하세요.
아래처럼 준비가 되면 다음 단계로 넘어가세요.
📷
step4_intro.png 으로 교체하세요
claude_project 폴더를 열면 이런 화면이 나와요
54-1과 4-2를 모두 설치하세요
4-1 (확장 프로그램)은 Antigravity 화면 안에서 Claude Code를 클릭으로 편하게 쓸 수 있게 해줘요. 앱 설치 같은 개념이에요.
4-2 (터미널)은 Claude Code를 컴퓨터 전체에 설치하는 거예요. 이게 있어야 핸드폰 연결, 다른 툴과 연동 같은 고급 기능들을 쓸 수 있어요. 드라이버 설치 같은 개념이에요.

→ 4-1만 하면 기본 기능만 되고, 4-2까지 해야 Claude Code의 모든 기능을 제대로 활용할 수 있어요!
터미널과 Antigravity, 뭐가 다른가요?
🖥️
터미널 — 검은 화면에 명령어를 직접 타이핑해서 컴퓨터에 지시하는 창구예요. 컴퓨터랑 문자로 직접 대화하는 곳이에요.
🏢
Antigravity — 터미널 기능을 포함하면서, 파일 탐색기 + 코드 편집기 + AI Agent까지 한 화면에 모아놓은 작업실 전체예요.
💡 4-2에서 "터미널로 설치"한다는 건 Antigravity를 버리는 게 아니라,
Antigravity 안에 있는 터미널 창을 열어서 설치하는 거예요!

4-1에서 설치를 했는데 왜 또 하냐고요?
터미널 방식으로 설치해야 Claude Code의 다양한 기능을 활용할 수 있어요.
예를 들어 Claude Code와 핸드폰 연결 같은 기능이 터미널 방식으로 쉽게 가능해요.
1
Antigravity의 Agent에게 '클로드 설치해줘' 라고 입력하세요
잠깐, Antigravity의 Agent가 뭔가요? 🤔
Antigravity 오른쪽에 있는 채팅창이에요. 여기에 말을 걸면 AI가 알아서 필요한 명령어를 실행해줘요.
터미널에 직접 명령어를 입력하는 대신, "클로드 설치해줘"처럼 말로 시키면 Agent가 대신 해주는 것이에요.
→ 코딩을 몰라도 AI에게 말로 작업을 시킬 수 있는 게 Antigravity의 핵심 기능이에요!
📷
step4_2_agent.png 으로 교체하세요
Agent에게 '클로드 설치해줘' 입력
2
run, run을 누르면서 읽어보면서 claude에 로그인하세요
읽어보면서 누르기 → AI 활용 연습하기. 이런 과정을 엄청 하게 됩니다!!

잠깐! 4-1, 4-2 둘 다 설치하셨나요?
그렇다면 터미널, 확장 프로그램으로 클로드 셋업 완료!

이제
1) 왼쪽 Claude 아이콘을 눌러 웹에서 Claude Code를 사용하거나
2) 터미널에서 claude를 불러서 코딩을 진행해보세요.

예시) OO폴더에 있는 파일의 제목을 날짜 기준으로 통일해서 만들어줘.
1번과 2번, 실제 차이가 있나요?
1) Claude 아이콘 (확장 프로그램)
· 채팅하듯 편하게 사용
· 현재 열린 파일/폴더만 인식
· 기능이 제한적
· 맛보기용
2) 터미널에서 claude ✦
· 프로젝트 전체를 통째로 이해
· 파일 생성·수정·실행 자동 처리
· 핸드폰 연결 등 고급 기능 사용
· 실무에서 쓰는 진짜 Claude Code
복잡한 작업일수록 터미널 방식이 훨씬 강력해요. 익숙해지면 터미널을 주로 쓰게 될 거예요 😎
